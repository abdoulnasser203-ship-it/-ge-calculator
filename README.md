# Age Calculator App

Un projet simple de calcul d'âge en HTML, CSS et JavaScript pur.

## Fonctionnalités

- Calcul de l'âge exact en années, mois et jours
- Validation des champs (champ vide, date invalide, date future...)
- Animation des chiffres au moment du résultat
- Design responsive (mobile et desktop)

## Structure du projet

```
age-calculator/
├── index.html        → Page principale
├── css/
│   └── style.css     → Tous les styles
├── js/
│   └── app.js        → Toute la logique JavaScript
└── README.md         → Ce fichier
```

## Comment utiliser

1. Ouvrir `index.html` dans un navigateur
2. Entrer le jour, le mois et l'année de naissance
3. Cliquer sur le bouton violet (ou appuyer sur Entrée)
4. L'âge s'affiche en années, mois et jours

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)
